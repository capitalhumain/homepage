Preface

主要都是Refernce裡那位在MacOX的作法，我在ubuntu再做一次，沒遇到啥問題

terence@terence-Latitude-E6410:~/Downloads$ sudo mv sbt-launch-0.7.7.jar /usr/local/bin/sbt-launcher.jar
[sudo] password for terence: 
terence@terence-Latitude-E6410:~/Downloads$ echo "java -Xmx512M -jar /usr/local/bin/sbt-launcher.jar \"\$@\"" | sudo tee /usr/local/bin/sbt
java -Xmx512M -jar /usr/local/bin/sbt-launcher.jar "$@"
terence@terence-Latitude-E6410:~/Downloads$ sudo chmod +x /usr/local/bin/sbt

然後就可以根據網頁的步驟create project來玩，記得如果是第一次執行要有心理準備需要從maven repository下載很多東西。
