Install Erlang

Environment

  • Ubuntu
  • Erlang R14B01

Context

改天要找一下乾淨的Ubuntu裝一次看看，因為這個開發用的ubuntu已經裝了很多有的沒的的東西了…，從Erlang網站抓最新release的source code自己build的作法如下。

在INSTALL.md裡面有列一堆需要預先安裝的東西，可以檢查一下。

  1. Unzip otp_src_R14B01.tar.gz
  2. ./configure
  3. make
  4. sudo make install

測試一下

terence@M1210:~$ erl
Erlang R14B01 (erts-5.8.2) [source] [smp:2:2] [rq:2] [async-threads:0] [hipe] [kernel-poll:false]

Eshell V5.8.2  (abort with ^G)
1>

Listing. basic.erl

-module(basic).
-export([mirror/1]).

mirror(Anything) -> Anything.

實際測試情形
