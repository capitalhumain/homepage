Using @EqualsAndHashCode

Listing 1. JavaDoc裡的範例
import groovy.transform.EqualsAndHashCode

@EqualsAndHashCode
class Person {
    String first, last
    int age
}

def p1 = new Person(first: 'John', last: 'Smith', age: 21)
def p2 = new Person(first: 'John', last: 'Smith', age: 21)
assert p1 == p2

def map = [:]
map[p1] = 45
assert map[p2] == 45

EqualsAndHashCode有提供三個影響產生的equals()和hashCode()運算方法的參數可以設定。

  • excludes
  • callSuper
  • includeFields
