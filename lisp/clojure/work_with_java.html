Environment

Using Java Classes

user=> (import '(java.util Calendar GregorianCalendar))
java.util.GregorianCalendar
user=> (. (new GregorianCalendar) get (. Calendar YEAR))
2011

第一行是import Calendar和GregorianCalendar。第二行程式就只是以下的程式碼(new GregorianCalendar()).get(Calendar.YEAR)在clojure的寫法。
