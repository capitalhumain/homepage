ECMAScript Object

最簡單建立javascript object的方法就是使用object literal。就是用大括號裡面放零到多個properties。 

var empty_object = {};

var peptide = {
    composition: [],
    toString: function() {
        return this.composition;
    }
};

這樣產生的object的constructor透過下列程式就可以知道是Object的constructor。 

alert( peptide.constructor );

問題來了當我們產生了一些物件，製作了一些有趣可重複使用的功能的程式，不管使用者有心或者無意之間很容易 因為變更了某些property而產生我們預期之外的行為。 

peptide.toString = function() {
    return 'n/a';
};

ECMAScript constructor and Prototype

ECMAScript並沒有像Java那類的語法支援Class的定義和宣告。ECMAScript是透過constructor來建立物件， 透過prototype property來做到物件繼承的功能。
